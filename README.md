# C++ OOP Practice - Multiple Objects

## Description
This program demonstrates the basic concepts of Object-Oriented Programming (OOP) in C++.

## Concepts Used
- Class
- Objects
- Data Members
- Creating Multiple Objects
- Accessing Members Using Dot Operator (.)

## Program Details
A class named `mobile` is created with:
- Brand (string)
- Price (float)

Two objects (`m1` and `m2`) are created to store and display information about different mobile phones.

Output:

The brand is: iphone
The price is: 10000
The brand is: Samsung
The price is: 50000
